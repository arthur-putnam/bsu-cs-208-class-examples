Fetch Example 1

This page will make an asynchronous fetch request to the Random User API when loaded. The JavaScript code in the `<script>` block initiates a fetch request to the endpoint 'https://randomuser.me/api/?results=1'. Upon receiving the response, it checks the status and parses the JSON data from the response body. If the status is 200 (OK), it logs the data to the console. If the status is not 200, an error message is logged. This example demonstrates how to fetch data from an external API using the Fetch API without reloading the entire webpage.
