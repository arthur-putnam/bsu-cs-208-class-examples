AJAX Example 1

This page will make an asynchronous AJAX request to the Random User API when loaded. The JavaScript code in the `<script>` block initiates an XMLHttpRequest (XHR) to the endpoint 'https://randomuser.me/api/?results=1'. Upon receiving the response, it checks the status and logs the JSON data to the console if the status is 200 (OK). If the status is not 200, an error message is logged. This example demonstrates how to fetch data from an external API without reloading the entire webpage.
